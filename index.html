FalconEye

FalconEye is a machine-learning prototype focused on detecting and classifying simulated aircraft system faults using sensor data. The project explores how data-driven models can assist in early fault identification, reduce pilot workload, and improve system reliability.

Key focus: feature extraction, fault classification, visualization, and ML fundamentals applied to aviation systems.

×

faultSense

faultSense is an experimental extension of fault detection ideas, aimed at understanding how predictive analytics and pattern recognition can be used to anticipate failures before they occur.

Focus: system thinking, data interpretation, and ML pipelines adapted for safety-critical aerospace domains.

Adobe React Spectrum

Contributed to Adobe’s React Spectrum design system by investigating and addressing a UI interaction bug where a button remained pressed after opening a modal.
